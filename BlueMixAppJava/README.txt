A data analytic web based application where events are retrieved as per location.  Based on the sentiment of the twitter data about the event, bar chart visualization and geographical map visualization were done using RESTful web services.  

APIs from IBM Bluemix and ESRI were used to create this application.

This project is created for a hackhathon conducted at "University of North Carolia, Chappel Hill".  We won the award "Best use of IBM Bluemix" for this project.
